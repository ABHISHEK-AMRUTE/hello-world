Installation
Windows
Download and install.

Mac
Download and install.

Linux
- Open a terminal.
- Add ppa:swi-prolog/stable to your system by running this command:
sudo add-apt-repository ppa:swi-prolog/stable

- Update the package, since it is often out of date:
sudo apt-get update

- Install SWI-Prolog:
sudo apt-get install swi-prolog

Running "Hello World"
Windows and Mac:
Open the hello_world.pl file.
Type hello_world. (With the period)

Linux:
Navigate to the directory containing hello_world.pl.
Type swipl (no quotes) to open SWI-prolog.
Type consult(hello_world).
Type hello_world.

Success! You ran your first Prolog program!
