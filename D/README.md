## Compiling and Running

`cd` to the folder that contains `hello-world.d`.

### Windows 

- Compile: `dmd hello-world.d`
- Run: `hello-world`

### Mac/Linux

- Compile: `$dmd hello-world.d`
- Run: `$ ./hello-world`
